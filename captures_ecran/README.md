# Banque d'images du projet
Ce dossier contient toutes les captures d'écran et images du projet Gestion de Projet Informatique.

## Organisation
Les images sont organisées par section du compte rendu :
- [A1], [A2], [A3] : Organisation

## Comment ajouter une image
1. Sauvegardez votre capture d'écran
2. Copiez-la dans ce dossier
3. Committez-la avec Git
4. Référencez-la dans le document LaTeX
