\subsection{[C2] Intégration continue}

        Nous avons mis en place un système d’intégration continue avec GitHub Actions.
        À chaque modification du projet, une vérification automatique est lancée pour s’assurer que le projet fonctionne correctement. Les erreurs de copilation et de code des fichiers Latex et du site web sont détéctées et signaléés. 
        Pour cela, nous avons utilisé l'outil d'intégration de version de github. Nous avons crée notre propre workflow car ceux prposés par GitHub Actions ne correspondaient pas à nos attentes.  
        Cela nous a permis de ne pas avoir à recompiler trop souvent ou toujours tester toutes nos contributions.
